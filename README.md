# React Burger

## Описание

Приложение "React Burger" позволяет пользователю собирать свой собственный бургер с различными ингредиентами. В приложении реализованы функции перетаскивания ингредиентов в конструктор бургера, а также возможность оформления заказа.

## Технологии

- **React** — библиотека для создания пользовательских интерфейсов.
- **Redux** — управление состоянием приложения.
- **Cypress** — для тестирования приложения.
- **React-DnD** — библиотека для реализации drag-and-drop функционала.
- **GitHub Pages** — для деплоя приложения на хостинг.

## Как скачать и запустить

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Exclusive048/react-burger.git
```

2. Перейдите в директорию проекта:

```bash
cd react-burger
```

3. Установите зависимости:

```bash
npm install
```

4. Запустите приложение:

```bash
npm start
```

Приложение будет доступно по адресу http://localhost:3000.
